# 컴포넌트

## 자주 쓰이는 기본 컴포넌트
| 컴포넌트               | 역할                      |
| ------------------ | ----------------------- |
| `View`             | HTML의 `div`처럼 레이아웃 컨테이너 |
| `Text`             | 텍스트 출력                  |
| `Image`            | 이미지 표시                  |
| `TouchableOpacity` | 터치 가능한 버튼               |
| `FlatList`         | 리스트 반복 출력               |
| `ScrollView`       | 스크롤 가능한 뷰               |
